### Type of strings
| Type        | Description   |
|-------------|---------------|
| "string"    | Double quotes |
| 'string'    | Single quotes |
| \`string\`  | Backticks     |
| <<<ABC ABC; | Heredoc       |

### String concatenation
| Type        | Description         |
|-------------|---------------------|
| .           | Dot                 |
| .=          | Dot equal           |
| ,           | Comma               |
| print       | Print               |
| printf      | Print with format   |
| var_dump    | 'string(5) "Hello"' |
* * * *
`printf("%s is a string, %d is a digit", $string, 2);`
* **%s** is a string, **%d** is a number
